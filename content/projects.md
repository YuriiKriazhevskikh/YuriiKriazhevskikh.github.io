### Test Automation for Flight Booking Platform
- **Stack:** Python, Pytest, Playwright, Requests, SQL, Azure DevOps
- Developed and maintained automated test suites for FareWise (web) and TravelWise (desktop) applications.
- Validated end-to-end booking workflows, GDS integrations (Amadeus, Sabre, Galileo), and database consistency.
- Integrated automated tests into CI/CD pipelines for daily regression runs.
- Performed performance testing with Apache JMeter, identified bottlenecks, and contributed to load balancer implementation.

### API Testing and Data Validation
- **Stack:** Python Requests, Postman, Swagger, SQL
- Created a comprehensive REST API test suite covering positive/negative scenarios, JSON/XML validation.
- Performed complex SQL queries to reconcile data between API responses and backend databases.

### Performance & Load Testing
- **Stack:** Apache JMeter, Grafana, Prometheus
- Designed and executed load tests to simulate high user concurrency, analysed metrics, and helped optimise system performance.
